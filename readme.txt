Este projeto foi desenvolvido para a H&W Publishing como parte de um teste de desenvolvimento front-end. A aplicação consiste em uma única página que inclui um formulário de captura de leads, seguido por uma apresentação de produtos após o envio bem-sucedido do formulário.

## 🚀 Funcionalidades Principais

- **Formulário de Captura de Leads**: Captura informações como nome, e-mail e telefone, com validação de entrada e máscaras para telefone.
- **Transição para Página de Produto**: Após o envio do formulário, o usuário é levado à página do produto com uma saudação personalizada.
- **Carrossel de Produtos**: Apresenta produtos em um carrossel dinâmico com opção de visualização em tela cheia.
- **Seções Interativas**: Inclui seções como Mais Vendidos, Feedback de Clientes e uma galeria de imagens interativa.

## 🛠️ Tecnologias Utilizadas

- **HTML5**
- **CSS3**
- **JavaScript Vanilla**
- **Bootstrap** para layouts responsivos
- **Swiper** para os sliders/carrosséis
- **Google Fonts** para tipografia

## 📚 Instruções de Instalação

1. Clone o repositório para seu ambiente local.
```bash
git clone https://github.com/hwpublishing/hw-single-page.git
```
2. Abra o arquivo `index.html` em um navegador moderno para visualizar a aplicação.
3. Certifique-se de estar conectado à internet para carregar os recursos externos, como Google Fonts e bibliotecas CSS/JS.

## 🎨 Interatividade e Animações

- Animações suaves para transições de seções.
- Carrossel dinâmico para navegação de produtos e feedback.
